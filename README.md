# O problema da equação do segundo grau
<br>

A equação do segundo grau é uma equação polinomial onde o termo de maior grau da variável é o quadrado. 
A sua forma geral é ax² + bx + c = 0, onde 'a', 'b' e 'c' são coeficientes (com 'a' ≠ 0) e 'x' é a variável. 
Para resolver uma equação do 2º grau, utiliza-se a fórmula de Bhaskara (fórmula resolvente) para encontrar as 
raízes da equação.
